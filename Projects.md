### Project-1: [AI-Powered Resume Screening System (NLP + DSA)] (https://github.com/blikhitha67/AI-Powered-Resume-Screener-DSA-NLP-.git)

An AI-driven system that automatically evaluates and ranks resumes against job descriptions using **NLP (TF-IDF + Cosine Similarity)** and **efficient data structures (HashMaps, Priority Queues, Tries)**. Trained on a Kaggle dataset, it achieves **96% accuracy** and provides scalable, interpretable results for recruitment.
